### RMI简介

**RMI(Remote Method Invocation,远程方法调用)**是java提供的分布式应用之间的协同调用能力，利用网络通讯，利用java的方式，实现分布式引用之间的通讯交互。

### 实现前的思考

（1）第一分布式应用必然涉及到网络通讯，第一个关键点**网络**

（2）数据在分布式应用直接数据如何传输与解析 第二个关键点**协议**

（3）如何是客户端获取到服务端可用的远程对象，第三个关键点**注册中心**
